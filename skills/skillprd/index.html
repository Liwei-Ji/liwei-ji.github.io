What It Does

SkillPRD transforms raw product ideas into professional, implementation ready Product Requirement Documents. It acts as a PRD Architect converting a minimal concept into a structured specification that engineers can build from and stakeholders can approve.

The skill operates in three phases:

  • Intake Discovery guides you through six core elements: Product Idea, Target User, Core Problem, Key Features, User Flow, and Success Goals
  • Delivery Target Selection choose from four output formats based on your audience
  • PRD Generation produces an 11 section document covering overview, KPIs, user flows, requirements, edge cases, and scope

Output formats: Clean Pro, Blueprint, UX Spec, and an investor grade HTML Pitch Deck.

Why I Built It

As a UX designer who works closely with product and engineering, I've written a lot of PRDs. The biggest friction I always noticed wasn't the writing itself it was the translation problem.

The same product idea needs to be communicated completely differently depending on who's reading it. An engineer wants acceptance criteria and edge cases. An investor wants the problem space and traction potential. A designer wants user flows and interaction states.

SkillPRD solves this by treating audience selection as a first class design decision not an afterthought.

I also wanted to explore what prompt engineering as product design looks like in practice. Designing a skill is not just writing instructions it's designing a conversation flow, handling ambiguous inputs, and making decisions about what the AI should and shouldn't do on the user's behalf.

How to Use It

$ step_01

Get the Skill

Go to the SkillPRD GitHub repository and open SKILL.md. Copy the entire file content.

$ step_02

Load into Your AI

Paste the content as a system prompt or custom instruction in:

  • Claude Project Instructions or system prompt
  • ChatGPT Custom Instructions or GPT configuration
  • Gemini Gems or system prompt

$ step_03

Start a Session

Tell the AI your product idea in plain language. It will guide you through the three phases automatically asking questions, confirming details, and generating the document in your chosen format.
